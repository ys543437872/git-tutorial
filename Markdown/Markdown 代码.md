# Markdown 代码
如果是段落上的一个函数或片段的代码可以用反引号把它包起来`` ` ``，例如：  
```
`printf()` 代码
```

显示结果如下：  
`printf()` 代码  

-------
## 代码区块
代码区块  
代码区块使用 4 个空格或者一个制表符（Tab 键）。  
实例如下：  
```
    Thist
    code
    block
```
显示结果如下：

    Thist
    code
    block

你也可以用```` ``` ````包裹一段代码，并指定一种语言（也可以不指定）：  
````
```c
int main(void)
{
    return 0;
}
```
````

显示结果如下：
```c
int main(void)
{
    return 0;
}
```
